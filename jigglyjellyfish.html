Jiggly Jellyfish

  • Category: Blockchain
  • Project date: August 2021
  • Project URL: opensea.io/collection/jigglyjellies

A NFT collection of 1,000 unique hand-drawn jellyfish

This project was inspired by Weird Whales, a NFT collection created by a 12-year-old coder in 2021. I hand-drew each element of the jellyfish (eyes, tentacles, body, accessories, mouth, etc.) and then wrote a Python program to randomly combine these elements into JPG images. Each jellyfish is unique given the way the program works, and they are available to purchase on the Ethereum blockchain. Prices range between 0.01 ETH and 0.1 ETH. All profits go towards oceana.org.
