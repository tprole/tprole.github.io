Road Sign Classifier

  • Category: AI
  • Project date: November 2021
  • Article: www.medium.com/@taraprole/road-signs-driving-you-crazy-e686d97f2480

A convolutional neural network capable of classifying road signs

This is a machine learning algorithm that I built in Python using PyTorch. Using the German Traffic Sign Recognition Benchmark (GTSRB) dataset, I was able to program a model that can classify road signs as one of 43 different classes. This project was inspired by my interest in autonomous vehicles and computer vision. To learn more about the project and how it was achieved, read the article above.
